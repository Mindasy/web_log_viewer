# 功能介绍

Web Log Viewer 是一个纯浏览器端的日志查看与分析工具，无需任何后端服务即可运行。所有解析、搜索、过滤操作均在本地完成，数据不会离开浏览器。

## 核心功能

### 多格式文件支持

支持多种文件和压缩格式的日志读取与解析：

- 直接读取 `.log`、`.txt`、`.csv`、`.json`、`.xml` 等文本文件
- 自动解压 `.zip`、`.tar`、`.tar.gz`、`.tgz`、`.gz` 压缩包，搜索其中的文本日志文件
- 支持多文件合并查看（可混合压缩包和普通文件）

### 日志解析引擎

内置 6 种日志格式预设，支持自动检测：

- **Bracket Log** — `[2026-01-15 08:00:00,000 +0800][INFO][1234][567][WEB-API][com.example.web.UserController] 用户 admin 登录成功`
- **Log4j / Logback** — `2026-01-15 08:00:00,000 INFO [http-nio-8080-exec-1] com.example.web.UserController: 用户 admin 登录成功`
- **Log4j2** — 支持带线程上下文的多字段格式
- **Syslog** — 标准 UNIX 系统日志格式
- **Apache / Nginx** — Web 服务器访问日志格式
- **JSON** — 自动映射常见字段名（timestamp、level、message 等）
- **通用时间戳** — 备用匹配，覆盖带时间戳的任意格式

同时也支持：

- **自定义正则** — 使用命名捕获组定义自己的解析规则
- **智能规则生成器** — 自动对样本行分词并推断字段分配，所见即所得
- **Pattern 持久化** — 将常用解析规则保存到浏览器 IndexedDB，跨会话复用

### 高性能虚拟滚动表格

使用虚拟滚动技术渲染日志表格，支持百万级日志行：

- 仅渲染可视区域的行（约 20-50 行），而非全量 DOM
- 列宽可拖拽调整，支持双击自适应
- 列排序（时间戳、级别、来源等）
- 列隐藏（右键菜单或设置面板）
- 自动隐藏空字段列
- 动态自定义字段列（JSON 日志自动展开为独立列）

### 全文搜索与过滤

- **实时搜索** — 输入即搜索，支持正则模式、大小写敏感、全词匹配
- **搜索结果高亮** — 匹配文本在表格中以黄色高亮显示，当前匹配以蓝色高亮
- **搜索结果导航** — F3 / Shift+F3 快速跳转上/下一个匹配
- **级别快速过滤** — 点击级别标签切换 FATAL / ERROR / WARN / INFO / DEBUG / TRACE 可见性
- **高级过滤** — 按线程、来源、消息内容、时间范围精确过滤
- **文件源过滤** — 在文件列表中点击文件，仅显示该文件中的日志行
- **过滤相似** — 在详情面板中，根据当前消息模式自动搜索相似条目
- **排序** — 点击列头升序/降序排列

### 智能跳转系统

跳转输入框支持多种快捷语法：

| 输入 | 跳转目标 |
|------|---------|
| `42` | 第 42 行 |
| `#5` | 第 5 个搜索结果 |
| `@3` | 第 3 个书签 |
| `$` | 末行 |
| `+50` | 向后 50 行 |
| `-20` | 向前 20 行 |

### 书签系统

- 双击行或 Ctrl+B 添加/删除书签
- 书签列表面板 — 查看所有书签、点击跳转、移除
- 书签导出 — 将书签列表导出为文件
- 清除所有书签

### 详情面板

在右侧面板中展示选中条目的每个字段值：

- 时间戳、级别、进程 ID、线程 ID、标签、来源、消息
- 自定义字段（JSON 日志或自定义正则自动提取）
- 每个字段独立复制按钮
- 操作按钮：复制整行、过滤相似、切换书签

### 统计信息

- **级别分布** — FATAL / ERROR / WARN / INFO / DEBUG / TRACE 柱状图
- **时间概览** — 日志时间跨度、平均速率
- **摘要** — 总条目数、错误数、文件大小、Top 线程、Top 来源

### 时间线可视化

Canvas 绘制的散点图时间线：

- 每个点代表一条日志，按级别颜色编码
- 滚轮缩放、拖拽平移
- 悬停显示行号和级别
- 点击跳转到对应日志行

### 主题切换

- **暗色主题（默认）** — Tokyo Night 风格，低蓝光护眼
- **亮色主题** — 白色背景，适合明亮环境
- 偏好设置通过 `localStorage` 持久化

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + O` | 打开文件 |
| `Ctrl + B` | 添加/删除书签 |
| `Ctrl + F` | 聚焦搜索 |
| `Ctrl + G` | 跳转到行 |
| `Ctrl + S` | 导出过滤结果 |
| `F3` | 下一个搜索结果 |
| `Shift + F3` | 上一个搜索结果 |
| `Enter` | 下一个搜索结果（搜索框聚焦时） |
| `Escape` | 关闭当前面板 |
| `↑ / ↓` | 上/下一行 |
| `PageUp / PageDown` | 上/下一页 |
| `Home / End` | 首行/末行 |
